# 🍎 Фруктовий Маркет (Tkinter) 🚚

Красивий та інтерактивний додаток для замовлення фруктів, написаний на Python з використанням бібліотеки **Tkinter** та графічного пакунка **Pillow**.

---

## ✨ Особливості проекту

* **🎨 Графічні іконки:** Замість стандартних чорно-білих системних емодзі використовуються яскраві кольорові іконки **Twemoji**.
* **🌓 Темна та Світла теми:** Зручний перемикач теми у верхньому правому куті, який повністю змінює оформлення додатку та вікон замовлення.
* **📦 Налаштування замовлення:** При кліку на фрукт відкривається модальне вікно, де можна вибрати колір за допомогою кольорових плашок (червоний, зелений, жовтий, оранжевий), вказати кількість та вписати ім'я отримувача.
* **✍️ Зручні поля введення:** Поле імені має зникаючий плейсхолдер (*зникає при фокусі та повертається, якщо поле пусте*).
* **🚚 Симуляція доставки:** Анімований статус замовлення, який показує процес підготовки, доставки та успішного отримання товару.

---

## 🛠️ Вимоги та встановлення

Для запуску проекту вам знадобиться **Python 3** та встановлена бібліотека **Pillow** для обробки зображень.

### 1. Клонування репозиторію
```bash
git clone https://github.com/greenyarik0505-jpg/Privet.git
cd Privet
```

### 2. Встановлення залежностей
```bash
pip install pillow
```

### 3. Запуск додатку
```bash
python main.py
```
