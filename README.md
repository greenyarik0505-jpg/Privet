# 🍎 Фруктовий Маркет (Tkinter Desktop Application) 🛒

Ласкаво просимо до **Фруктового Маркету** — сучасного настільного додатка для замовлення та купівлі свіжих фруктів, написаного на мові програмування Python з використанням бібліотеки **Tkinter** та обробника зображень **Pillow**.

Цей проект демонструє можливості створення інтерактивних графічних інтерфейсів (GUI) користувача з підтримкою динамічних стилів, обробкою подій, пошуком, історією замовлень, генерацією паперових чеків, управлінням станом кошика, інтеграцією промокодів та валідацією форм.

## ✨ Особливості проекту

* **🖼️ Графічне оформлення:** Кожна картка товару містить барвисту іконку фруктів високої якості (на базі відомого дизайну Twemoji), які автоматично ресайзяться під роздільну здатність екрану.
* **🔍 Живий пошук:** Поле пошуку вгорі екрана фільтрує фрукти за назвою в реальному часі під час введення літер.
* **🌈 Інтерактивний вибір кольору/сорту:**
  * Замість стандартних текстових списків реалізовано палітру кольорових кнопок-плашок безпосередньо у вікні кожного товару.
  * Вибір кольору адаптується під конкретний фрукт (наприклад, для винограду доступні Синій 🔵, Зелений 🟢, Фіолетовий 🟣, а для кавуна — Смугастий 🟢 або Темний 🟢).
* **🔢 Гнучке регулювання кількості:** Зручний вибір кількості одиниць товару через компонент `Spinbox` (з підтримкою валідації значень).
* **📜 Історія замовлень:** Вікно історії сесії показує всі успішно оформлені замовлення, кількість товарів та витрачені суми.
* **📄 Друк паперових чеків:** При кожному оформленні замовлення автоматично генерується і зберігається на диск текстовий файл чеку з детальною розшифровкою покупки (дати, ціни, знижки та імені отримувача).
* **🛒 Просунуте управління кошиком:**
  * Усі додані позиції зберігаються у списку з точним зазначенням сорту (кольору) та кількості.
  * Можливість видалення окремих позицій (кнопка ❌) або повне очищення кошика.
* **🎟️ Дисконтна система промокодів:**
  * Вбудована перевірка промокодів. Застосування коду **`FRUIT20`** автоматично розраховує знижку **20%** на загальну суму та миттєво перераховує підсумковий чек.
* **🔒 Валідація замовлення:**
  * Інтерактивне вікно оформлення доставки запитує ім'я отримувача.
  * Поле вводу має зникаючий плейсхолдер (*зникає при фокусуванні мишкою та повертається, якщо поле залишене порожнім*).
  * Передбачена перевірка імені на коректність, що виключає випадкове замовлення на дефолтний текст.

## 🎨 Специфікація інтерфейсу та дизайну

Графічний інтерфейс побудовано з урахуванням сучасних стандартів UX/UI:
* **Шрифти:** Використовується системний шрифт `Segoe UI` (або `Segoe UI Emoji`), що забезпечує чіткість тексту та згладжування символів.
* **Адаптивність вікон:** Всі діалогові та допоміжні вікна мають фіксовані розміри, центруються відносно батьківського вікна та є модальними (`grab_set()`), що блокує випадкові подвійні натискання на головному екрані під час вибору.
* **Колірна гама:**
  * Фон додатку: нейтральний світло-сірий (`#f8f9fa`).
  * Кнопки дій: акцентний синій (`#4a90e2`), успішне зелене підтвердження (`#2ecc71`), попередження/видалення (`#ff4d4d`), темний кошик (`#2c3e50`).

## 🧩 Архітектура даних та функцій

### Структура даних фруктів (`fruits_data`):
```python
fruits_data = {
    "Яблуко 🍎": {
        "price": 20, 
        "desc": "Соковите червоне або зелене яблуко", 
        "img": "apple.png",
        "colors": [("Червоне 🔴", "#e74c3c"), ("Зелене 🟢", "#2ecc71"), ("Жовте 🟡", "#f1c40f")]
    },
    # Інші товари...
}
```

### Основні функції:
* `open_details(name)`: Створює модальне вікно `Toplevel` для налаштування параметрів фрукта (вибір сорту, кількості).
* `add_to_cart(name, price, qty, color, dialog)`: Проводить валідацію введених даних, групує замовлення за назвою та кольором, оновлює інтерфейс кошика.
* `view_cart()`: Відкриває панель кошика, створює динамічний список товарів з кнопками видалення та керує промокодами.
* `refresh_cart_view()`: Перераховує ціни та перебудовує віджети всередині фрейму кошика при будь-якій зміні складу покупок.
* `filter_fruits()`: Зчитує текст із поля пошуку та оновлює сітку карткових елементів у реальному часі.

## 🛠️ Встановлення та запуск

### Системні вимоги:
* Операційна система: Windows 10/11, macOS або дистрибутиви Linux.
* Версія Python: **3.8 або вище**.
* Встановлений пакетний менеджер `pip`.

### Крок 1. Клонування репозиторію
Склонуйте репозиторій до себе на локальний диск:
```bash
git clone https://github.com/greenyarik0505-jpg/Privet.git
cd Privet
```

### Крок 2. Встановлення Pillow
Цей додаток використовує бібліотеку **Pillow (PIL)** для якісного масштабування картинок фруктів:
```bash
pip install pillow
```

### Крок 3. Запуск додатку
Виконайте скрипт за допомогою Python:
```bash
python main.py
```

## 📖 Інструкція користувача

1. **Вибір товару:** На головному екрані виберіть будь-який фрукт та натисніть під ним кнопку **"Детальніше"**.
2. **Налаштування параметрів:**
   * Оберіть бажаний колір/сорт, натиснувши на одну з кольорових кнопок у вікні.
   * Виберіть кількість за допомогою стрілочок або введіть число вручну.
   * Натисніть **"Додати в кошик"**.
3. **Перегляд та редагування кошика:**
   * Натисніть на велику кнопку **"🛒 Кошик"** внизу головного екрана.
   * Для видалення зайвого товару натисніть кнопку **❌** праворуч від його назви.
   * Для отримання знижки введіть у текстове поле промокод `FRUIT20` та натисніть кнопку **"Застосувати"**.
4. **Оформлення замовлення:**
   * Натисніть **"Оформити"**.
   * У спливаючому вікні клікніть на текстове поле (напис "Ваше ім'я" зникне автоматично), введіть своє ім'я та натисніть **"Підтвердити замовлення"**.
   * Текстовий чек буде збережено у кореневу папку проекту у форматі `receipt_YYYYMMDD_HHMMSS.txt`.

## 💻 Інструкція для розробників

Якщо ви хочете розширити асортимент магазину, ви можете зробити це без зміни логіки інтерфейсу.

### Як додати новий фрукт:
1. Завантажте картинку нового фрукта у форматі PNG (бажано квадратної форми) та помістіть її у кореневу папку проекту.
2. Відкрийте файл `main.py` та додайте новий елемент у словник `fruits_data`:
   ```python

       "price": 35,
       "desc": "Солодка стигла груша дюшес",
       "img": "pear.png",
       "colors": [("Жовта 🟡", "#f1c40f"), ("Зелена 🟢", "#2ecc71")]
   }
   ```
3. Запустіть додаток — картка створиться та налаштується автоматично!

## ❓ Часто задавані питання (FAQ)

**Q: Що робити, якщо виникає помилка `ModuleNotFoundError: No module named 'PIL'`?**  
A: Переконайтеся, що ви встановили бібліотеку Pillow у правильне середовище за допомогою команди `pip install pillow`. Якщо у вас встановлено кілька версий Python, спробуйте запустити команду `python -m pip install pillow`.

**Q: Чому деякі картинки не відображаються?**  
A: Перевірте, чи знаходяться файли зображень (`apple.png`, `banana.png` тощо) в тій самій папці, що й файл `main.py`. Назви файлів повинні точно збігатися з тими, які прописані в словнику `fruits_data`.
