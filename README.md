# 🍎 Фруктовий Маркет (Tkinter) 🛒

Простий та зручний настільний додаток для вибору фруктів та оформлення покупок, написаний на Python з використанням бібліотеки **Tkinter**.

---

## ✨ Особливості проекту

* **📦 Асортимент товарів:** Відображення списку свіжих фруктів з цінами та детальним описом.
* **🔍 Деталі товару:** При натисканні на фрукт відкривається окреме вікно з описом та кнопкою додавання до кошика.
* **🛒 Функціональний кошик:** Можливість переглядати список доданих товарів, рахувати загальну вартість замовлення та підтверджувати покупку.
* **💻 Простий інтерфейс:** Лаконічний дизайн, який легко запускається на будь-якому комп'ютері без додаткових налаштувань.

---

## 🛠️ Встановлення та запуск

Проект використовує виключно вбудовану бібліотеку **Tkinter**, тому додаткові залежності встановлювати не потрібно.

### 1. Клонування репозиторію
```bash
git clone https://github.com/greenyarik0505-jpg/Privet.git
cd Privet
```

### 2. Запуск додатку
```bash
python main.py
```
