# 🍎 Фруктовий Маркет (Tkinter Shopping App) 🛒

Сучасний, барвистий та багатофункціональний настільний додаток для покупки свіжих фруктів. Написаний на Python з використанням бібліотеки **Tkinter** та бібліотеки **Pillow** для обробки та відображення зображень.

---

## ✨ Основні можливості (Features)

1. **🖼️ Візуальний інтерфейс:**
   * Карткова сітка товарів з реальними кольоровими зображеннями фруктів (на базі Twemoji).
   * Інтегроване відображення фотографій у картках товарів та у спливаючому вікні деталей.

2. **🔍 Картка детальної інформації:**
   * При натисканні на кнопку "Детальніше" відкривається окреме модальне вікно.
   * У ньому відображається збільшена фотографія фрукта, опис, актуальна ціна за штуку та вибір кількості.

3. **🔢 Вибір кількості товарів (Quantity Selector):**
   * Ви можете додати будь-яку кількість обраного товару за допомогою зручного віджета `Spinbox` (від 1 до 50 шт.).

4. **🛒 Розширений кошик покупця (Advanced Cart Management):**
   * Інтерактивне вікно кошика, що відображає список усіх замовлених фруктів, кількість та загальну вартість за кожен вид.
   * **Поштучне видалення:** Можна легко видалити будь-яку конкретну позицію з кошика за допомогою кнопки ❌ навпроти неї.
   * **Очищення кошика:** Спеціальна кнопка "Очистити кошик" для повного скидання замовлення в один клік.

5. **🎟️ Система промокодів (Promo Code System):**
   * Можливість ввести спеціальний промокод у кошику.
   * Використовуйте промокод **`FRUIT20`**, щоб отримати миттєву **знижку 20%** на всю суму замовлення!

6. **🚚 Оформлення доставки:**
   * Після натискання кнопки "Оформити" замовлення автоматично переходить на стадію доставки із виведенням візуального повідомлення покупцю.

## 🛠️ Встановлення та запуск

Для роботи додатка потрібен встановлений інтерпретатор Python 3, а також графічна бібліотека **Pillow**.

### 1. Клонування репозиторію
```bash
git clone https://github.com/greenyarik0505-jpg/Privet.git
cd Privet
```

### 2. Встановлення Pillow
Бібліотека Pillow використовується для завантаження, зміни розміру та відображення PNG-зображень фруктів:
```bash
pip install pillow
```

### 3. Запуск проекту
Запустіть додаток командою:
```bash
python main.py
```
